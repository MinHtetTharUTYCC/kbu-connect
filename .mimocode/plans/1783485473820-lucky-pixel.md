# Plan: Change App Font to Inter

## Recommendation

Replace **Geist** with **Inter** (sans) and **Geist Mono** with **JetBrains Mono** (mono).

**Why Inter?**

- The most popular modern UI font, used by Vercel, GitHub, Linear, and thousands of apps
- Optimized for screens with excellent readability at all sizes
- Variable font with fine-tuned weight control
- Neutral/professional feel that suits a university app

**Why JetBrains Mono?**

- Distinctive, highly readable monospace font
- Popular among developers, modern aesthetic
- Ligature support for a polished look

## Changes

### 1. `src/app/layout.tsx`

- Replace `Geist` import with `Inter` from `next/font/google`
- Replace `Geist_Mono` import with `JetBrains_Mono` from `next/font/google`
- Update variable names and font configuration

### 2. `src/app/globals.css`

- No changes needed — the `--font-sans` and `--font-mono` CSS variables are already mapped via the `variable` property in `layout.tsx`, and the fallback stacks in `globals.css` serve as safety nets

## Verification

- Run `pnpm run build` (or `next build`) to ensure no errors
- Run `pnpm run dev` and visually confirm the new font renders correctly across pages
