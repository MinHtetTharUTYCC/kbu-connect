# Plan: Full Profile View on Discovery Page

## Summary
Add a bottom sheet modal that shows the full user profile when tapping a discovery card's name.

## Trigger
User taps `profile.name` on the discovery card → opens a bottom sheet with the full profile.

## Implementation

### 1. Create `ProfileSheet` component
**File:** `src/components/mobile/profile-sheet.tsx`

A bottom sheet (similar to `ShoutoutSheet`) that:
- Fetches full profile via `useVisitProfile(userId)`
- Shows loading skeleton while fetching
- Displays: gallery grid (3-col, first spans 2), name, age, faculty, nationality, bio, interests
- Action buttons at bottom: Like / Pass / Send shoutout
- Close button (X) in top-right
- z-index: `z-[70]` (above ShoutoutSheet z-[60], below FullScreenImageViewer z-[100])

Props:
```ts
{
  userId: string;
  onClose: () => void;
  onLike: () => void;
  onDislike: () => void;
  onShoutout: () => void;
}
```

### 2. Update `discover-client.tsx`
**File:** `src/app/(web)/discover/_components/discover-client.tsx`

- Make `profile.name` a clickable button
- Add `selectedProfileId` state (string | null)
- When name is tapped → `setSelectedProfileId(profile.id)`
- Render `<ProfileSheet>` when `selectedProfileId` is set
- Wire Like/Pass/Shoutout actions from the sheet to the existing handlers
- Close sheet on action or X button

### 3. Gallery images in the sheet
- Fetch from the full profile response (which includes `gallery[]`)
- Render same grid layout as `profile-client.tsx` (first image col-span-2)
- Tap on gallery image → open `FullScreenImageViewer`
